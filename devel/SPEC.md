# 仕様書 (SPEC) - 15色投票アプリケーション

本アプリケーションは、Apacheサーバー上で動作する、15色のカラー選択および投票（「好き」「嫌い」）を行える高機能なシングルページWebアプリケーション（SPA）です。

## 1. 動作環境・構成
- **サーバー**: Apache (静的ファイル配信)
- **格納ディレクトリ**: `/workdir/project0002/html`
- **構成ファイル**:
  - `index.html` (構造・画面遷移レイアウト)
  - `styles.css` (モダンでレスポンシブなVanilla CSS)
  - `app.js` (Vanilla JSによるステート管理、インタラクション、LocalStorage永続化)

## 2. 機能要件

### Phase 1: カラー選択画面 (15色選定)
- **対象カラー (18色)**:
  1. 赤 (Red)
  2. 青 (Blue)
  3. 緑 (Green)
  4. ピンク (Pink)
  5. オレンジ (Orange)
  6. イエロー (Yellow)
  7. ブラック (Black)
  8. ホワイト (White)
  9. 紫 (Purple)
  10. ブラウン (Brown)
  11. シアン (Cyan)
  12. ライム (Lime)
  13. マルーン (Maroon)
  14. ローズ (Rose)
  15. バナナ (Banana)
  16. グレー (Gray)
  17. タン (Tan)
  18. コーラル (Coral)
- **選択動作**:
  - ユーザーは18色の中から**重複なしで正確に15色**を選択します。
  - カラーカードをクリックすることで選択・解除をトグルできます。
  - 現在の選択数が「選択数: X / 15」のようにリアルタイム表示されます。
  - 15色未満、あるいは15色を超えている場合は「投票画面に進む」ボタンが非活性 (disabled) になります。
  - ちょうど15色選択されたときのみ、ボタンが活性化し、Phase 2に進めます。

### Phase 2: 投票画面 (「好き」「嫌い」)
- **表示内容**:
  - Phase 1で選択された15色のみが、洗練されたグリッドレイアウトで表示されます。
- **投票モード切替ボタン**:
  - 「好き (Like)」ボタン（赤基調、アクティブ時は赤く発光・ハイライト）
  - 「嫌い (Dislike)」ボタン（青基調、アクティブ時は青く発光・ハイライト）
  - 初期状態は「好き」モードがアクティブになっています。
- **カラーボタンのクリック動作**:
  - 投票モード（「好き」または「嫌い」）が選択されている状態で、15色のカラーボタンのいずれかをクリックすると、その色に投票が記録されます。
  - **「好き」モードがアクティブなとき**:
    - クリックされた色が「未投票」または「嫌い」であれば、「好き」マーク（赤いハートマーク ❤️ またはカスタム赤バッジ）が付きます。
    - すでに「好き」マークが付いている場合は、クリックによりマークが解除されて「未投票」に戻ります（トグル動作）。
  - **「嫌い」モードがアクティブなとき**:
    - クリックされた色が「未投票」または「好き」であれば、「嫌い」マーク（青い割れたハート 💔 またはカスタム青バッジ）が付きます。
    - すでに「嫌い」マークが付いている場合は、クリックによりマークが解除されて「未投票」に戻ります（トグル動作）。
- **データ保存 (localStorage)**:
  - 選択された15色、各色の投票状態（'like', 'dislike', 'none'）、現在のフェーズ、現在の投票モードが、操作のたびに自動的にブラウザの `localStorage` に保存されます。
  - ページをリロードしても、直前の状態が完全に復元されます。
- **リセット・再選択機能**:
  - 「カラーを選び直す」ボタンを提供。クリックすると確認ダイアログが表示され、承諾すると現在の投票データおよび選択カラーをリセットして Phase 1 に戻ることができます。
  - 「すべてクリア」ボタンを提供。クリックすると投票状態のみを全てクリア（未投票状態にリセット）します。

## 3. デザイン・UI/UX方針 (Vanilla CSS)
- **テーマ**: Modern & Clean Soft-UI
- **配色**: 
  - 背景: 薄いグレー (`#f8f9fa`)
  - カード: 純白 (`#ffffff`)、ソフトなシャドウ (`box-shadow: 0 4px 15px rgba(0,0,0,0.05)`)
  - 各カラーチップは、実物の色を美しく表現するためのカラーパレット（CSS変数）を厳密に定義して実装（白は黒い縁取りをつけるなど視認性を確保）。
- **フォント**: システムフォント (`system-ui, -apple-system, sans-serif`)
- **レスポンシブ対応**: スマートフォン、タブレット、PCそれぞれの画面幅に合わせて、CSS GridとFlexboxを用いて最適化（グリッド数が動的に変化）。
- **トランジション**: ボタンのホバー、アクティブ、カード選択時のスケール変化に滑らかなアニメーション (`transition: all 0.2s ease`) を設定。
