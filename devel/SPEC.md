# 仕様書 (SPEC) - 高機能多段階カラー投票・相性検証アプリケーション

本アプリケーションは、Apacheサーバー上で動作する、18色から選定された15色のカラーに対する個別評価（好み）、時系列評価（脱落・除外）、全共通属性（ダメ判定・メモ機能）、およびカラー間の双方向相性タグ（合う・合わない）を網羅的に検証・記録できる高機能なシングルページWebアプリケーション（SPA）です。

## 1. 動作環境・ファイル構成
- **サーバー**: Apache (静的配信環境)
- **格納ディレクトリ**: `/workdir/project0002/html`
- **構成ファイル**:
  - `index.html`: セレクター構造、3カラムUI、2段階アコーディオンを定義する骨組み要素。
  - `styles.css`: Soft-UIを基調としたモダンなデザインシステム。半透明、点線、ブラー、パルスアニメーションによる状態変化を制御。
  - `app.js`: シングル永続ステート（LocalStorage）の管理、多段階ソートアルゴリズム、およびイベントバブリング遮断を伴う動的DOM制御ロジック。

## 2. 画面レイアウト・機能要件

### Phase 1: カラー除外選定画面
- **概要**: 18色のカラーバリエーションカードから、分析に**不要な3色**を正確に除外します。
- **視覚的特徴**: 
  - 除外対象としてクリックされたカードは、背景がトーンダウンし、カラースウォッチ（色丸）の不透明度が大幅に低下（点線枠化）します。
  - カード右上へ、パルスアニメーション（緩やかに脈打つ効果）を持った鮮やかな赤色の「✖ 除外」バッジが出現し、選択状態を直感的に明示します。
  - ちょうど3色が選択されたときのみ、下部の「投票画面に進む」ボタンが活性化します。

### Phase 2: メイン投票・相性検証画面 (3カラムレイアウト)
Phase 1を通過した15色のカラーを軸に、右サイドバーの「ラウンドメニュー（第1回、第2回…）」を切り替えながら、多角的な検証を行います。

#### ① 左サイドバー: タグソース
- 生存している15色のミニカラーリスト。
- メインエリアで「合う」「合わない」モードを使用する際、起点（ソースカラー）として機能します。
- 「ダメ」「脱落」「除外」モードがアクティブな間は、操作ミス防止のため自動的に半透明（非活性化）になり、選択できない安全ガードが機能します。

#### ② 中央メインエリア: 投票グリッド・モードセレクター
最上部に、コンパクトに整理された2階層対応の「モードセレクター」が配置されています。

##### 【モードセレクターの仕様】
- **好み▼ (Preference)**: 
  - クリックすると、下部に「好みサブパネル（役職持ち / エンジニア / 科学者 / 探偵 / トラッカー / ノイズメーカー等）」が滑らかにスライド展開します。
  - いずれかの好みが選ばれている間はパネルが開いた状態をキープし、他のメインモード（ダメ、脱落等）がクリックされると自動で美しく格納されます。
- **ダメ (No)**: 全ラウンド共通の絶対不可タグ。
- **脱落 (Drop)**: 指定ラウンド以降に適用される準除外ステータス。
- **除外 (Exclude)**: 指定ラウンド以降に適用される完全除外ステータス。
- **合う (Fit) / 合わない (Not Fit)**: ラウンドごとのカラー相性リンクモード。

##### 【中央カラーカードの動的レイアウト・ソート仕様】
各カラーカードは、上から **「共通ステータスタグ」 ＞ 「左:好みタグ / 中:色丸と名前」 ＞ 「下:相性タグリンク」 ＞ 「最下部:メモ欄」** という厳格な視覚階層で構成され、**現在のラウンド数に応じて以下の優先度でリアルタイムに自動ソート（並び替え）**されます。

1. **通常カラー（最上部へ配置）**
   - 評価タグや相性リンクがクリアに見やすく並びます。
2. **脱落カラー（中間下部へ自動配置）**
   - 「脱落」モードでクリックすると、**そのラウンド以降のすべての画面**において自動で下部に並び替わります。
   - カード背景が半透明化（不透明度 0.4）、スウォッチが彩度90%ダウン（モノクロ化）し、カード上部へ「脱落 🏴 (第X回～)」タグが表示されます。設定された回より過去のラウンド画面に戻ると、通常カラーの位置へ自動で復帰します。
3. **除外カラー（最下部へ自動配置）**
   - 「除外」モードでクリックすると、**そのラウンド以降のすべての画面**において自動で最も下に沈み込みます。
   - カード背景のグレーアウトが強まり、不透明度が 0.25 まで低下、スウォッチには100%モノクロ化＋ブラー（微細なぼかし）効果が適用され、完全に戦力外であることが強調されます。カード上部には「除外 🚫 (第X回～)」タグが表示されます。

##### 【独立イベントメモエリアの仕様】
- カード最下部に配置されたテキストエリア（メモ欄）は、**全ラウンド共通でリアルタイム保存**されます。
- 文字入力やスクロール、マウスダウン、キーボード打鍵のあらゆる瞬間において、カード本体へのイベント伝達（バブリング）を JavaScript 側で完全に遮断（`stopPropagation`）しています。
- そのため、上部でどのような投票・脱落・除外モードがONになっていても、**タグの誤暴発を起こすことなく安全に文字入力を編集・記述することが可能**です。

#### ③ 右サイドバー: ラウンドメニュー
- 「次の回を作成」を押すことで、第1回、第2回…と新しい検証シートを無限に拡張できます。
- シートを切り替えても、「好み」「ダメ」「メモ」、そして「指定回以降の脱落・除外」の状態は内部データ（`localStorage`）で完全に一貫性を保って同期・保護され、シートごとの純粋な相性タグの差異のみを快適に比較追跡できます。
